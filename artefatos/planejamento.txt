DESCREVER EM ITENS AS FERRAMENTAS/TECNOLOGIAS SERÃO UTILIZADAS NO STACK TECNOLÓGICO E O MOTIVO DO SEU USO
* Trelo -> Vai ser usado para gerenciar as atividas;
* Github -> Usado por ser uma plataforma aberta, que todos conhecemos e para fazer o versionamento;
* PHP -> Usado no server-side, por ser uma linguagem já estabelecida e ser amplamente usada neste tipo de sistemas web;
* Laravel -> Usado por ser um framework MVC bastante popular e com uma serie de recursos úteis;
* HTML, CSS e JS -> Usado para a criação do front-end;
* Bootstrap -> Usado pela sua facilidade e agilidade para desenvolver o front-end;
* Nginx -> Servidor web utilizado, estamos usando por ser bastante eficiente e difundido;
* PostgreSQL -> Usado por ser um banco bastante utilizado, por atender bem a nossa demanda;

DESCREVER QUAL A METODOLOGIA SERÁ UTILIZADA NO PROJETO E O PORQUÊ

Optamos por usar o modelo cascata pois vai ser um projeto pequeno, já levantamos os requisitos, funcionalidades, já fizemos uma pesquisa de opinião e temos uma ideia fechada já de como vai ficar a versão inicial.
